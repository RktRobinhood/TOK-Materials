Test claims like a scientist. Use falsification.

Investigate a climate-related claim using peer‑reviewed research. Build a clear falsification test: What evidence would prove the claim wrong? Present your findings in a 10–12 slide deck.

Required falsification slide

  • Prediction: If our claim is true, we should observe ____.
  • Falsifier: If we instead observe ____, the claim would be wrong or need revision.
  • Dataset: Which measurable data could test this?
